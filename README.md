# Угадай число на Python

![Угадай число на Python](img/file.png "Игра 'Угадай число'")

## Описание программы

Это простая консольная игра **"Угадай число"**, написанная на языке программирования Python. Компьютер загадывает случайное число, а игрок должен его отгадать за ограниченное количество попыток. :blush:

### Необходимые условия для запуска проекта

1. Редактор кода (рекомендуется VS Code)
1. Язык программирования Python версии 3.6+
1. Операционная система Windows, Linux или Mac
1. ~~_Mysql_~~

| Шаг | Установка | Ссылка |
|-----|-----------|---------|
| 1 | Скачать установочный файл редактора кода **под вашу ОС** и установить на ваш ПК | [VS Code](https://code.visualstudio.com/) |
| 2 | Скачать установочный файл языка программирования **под вашу ОС** и установить на ваш ПК | [Python](https://www.python.org/) |

> Вся логика в коде стоится на библеотеке random. :heart:
